## ⚙ Get Started

1. Clone project

    ```bash
    git clone {GIT_LINK}
    ```

2. Go to project folder

    ```bash
    cd general-web-compo
    ```

3. Set up environment by command

    ```bash
    yarn

    # or use `yarn install`
    ```

4. Run project by command

    ```bash
    # Normal Mode
    yarn dev

    # Build Mode
    yarn build
    ```
